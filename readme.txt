
Hi, this is a repository for fancy tools in houdini. Please, use it and give me feedback.

1st Step, edit you hou.env

HOUDINI_PATH = USERPATH\GitHub\fxTools;&

Example:
HOUDINI_PATH = C:\Users\AiraM\Documents\GitHub\fxTools;&

Also, you need to create a ENV Variable (windows user)

variable ---> PYTHONPATH 
path ---> " GITPATH\GitHub\fxTools; "

Example of path ---> C:\Users\AiraM\Documents\GitHub\fxTools;
If you already have a PYTHONPATH created, you can add after the old path with ' ; '
