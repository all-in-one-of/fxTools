Muy buenas, para empezar, este va a ser un repositorio de tools apra houdini, asi que a darle caña, y a comentar ideas y fallos que vean.

Para empezar, tienen que bajarse la APP de gitHub Desktop o descargarse los repositorios, y luego modificar el .env de houdini y añadirle la ruta donde tienen guardado el repositorio

Como ha de ser:
HOUDINI_PATH = USERPATH\GitHub\fxTools;&

Ejemplo:
HOUDINI_PATH = C:\Users\AiraM\Documents\GitHub\fxTools;&

Tambien tienen que crearse una variable de entorno, es muy facil:

(doy por supuesto que todos usamos windows)
En el buscador de windows, pongan "Editar variables del sistema", luego pulsan en "Variables de entorno", y luego de crean uma nueva variable de nombre PYTHONPATH, y valor de la variable " RUTADELGIT\GitHub\fxTools; "

Ejemplo del nombre de la variable: C:\Users\AiraM\Documents\GitHub\fxTools;
Si ya tienen una variable de entorno llamada PYTHONPATH, pongan la ruta seguido del punto y coma " ; " y no habra ningun problema
